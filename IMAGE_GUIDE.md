# 🔥 火加減（HIKAGEN）画像管理ガイド

## フォルダ構成

```
hikagen/
├── index.html          ← アプリ本体
└── images/
    ├── opening.png     ← オープニング挿し絵（1枚）
    ├── s1-1.png        ← Scene1 バリエーション1
    ├── s1-2.png        ← Scene1 バリエーション2
    ├── s1-3.png        ← Scene1 バリエーション3
    ├── s2-1.png
    ├── s2-2.png
    ├── s2-3.png
    ├── ...             ← s3〜s8 も同様
    ├── s9-1.png
    ├── s9-2.png
    ├── s9-3.png
    └── ending.png      ← エンディング挿し絵（1枚）
```

## ファイル名規則

| 用途 | 命名規則 | 例 |
|------|----------|-----|
| オープニング | `opening.png` | `opening.png` |
| シーン画像 | `s{シーン番号}-{バリエーション}.png` | `s1-1.png`, `s5-3.png` |
| エンディング | `ending.png` | `ending.png` |

- シーン番号：1〜9（固定）
- バリエーション：1〜3（ランダム表示用）
- **合計：29枚**（opening 1枚 + 9シーン×3枚 + ending 1枚）

## 画像仕様（仕様書より）

- 水彩＋鉛筆風
- 寒色基調（暖色は火のみ）
- 性別・年齢不詳
- 表情は描き込みすぎない
- 宗教・神秘表現なし
- テキスト・記号なし
- 推奨サイズ：**横幅 800px**（モバイル表示用に十分）
- 形式：PNG（透過不要ならJPGでも可）
- アスペクト比推奨：**4:3 または 3:2**

## 画像がまだない場合

プレースホルダーが自動表示されるので、
画像を用意してからフォルダに入れるだけでOKです。
